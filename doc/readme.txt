Optionen der Unitext-Klasse:

Titelseite/Art der Arbeit:
  - bachelorarbeit - projektarbeit - masterarbeit
  - seminar        - praktikum     - teamprojekt
  - studienarbeit  - diplomarbeit  - bericht 
  - script         - dissertationv (eingereichte Fassung)
  - dissertationn (angenommene Fassung)

Titelseite/Institute:
  - inf  - ips  - idb  - iti  - ida  - eis  - sse

Literaturverzeichnis:
  - abbrv  - alpha  - plain  - unsrt

Sprachen:
  - english  - german

Ausgabeformat:
  - pdf  - ps  - dvi

Beispiele für Titelseite:
  - titelseite - cd

Default:
  - inf  - plain

------------------------------------------------------------

Zur Info: Bestandteile einer Titelseite:

  - \titlehead - \subject      - \title   - \author
  - \date      - \publishers   - \and     - \thanks
